## The Problem

1. People record voice notes on the go but 80%+ are never actioned — they die in WhatsApp/Telegram as unlistened memos
2. Voice-to-text apps (Otter, Whisper alone) only transcribe — they don't structure, prioritize, or create tasks
3. Typing tasks into Notion while driving, walking, or mid-meeting is impossible — ideas are lost before you reach a keyboard
4. Project management tools require manual input — there's no frictionless voice-first capture workflow
5. Existing tools like Siri Reminders and Google Assistant create basic reminders, not rich, categorized project tasks with priority, tags, and project context

## The Solution

A Telegram bot + n8n workflow that:
- Listens for voice messages sent to a Telegram bot
- Calls Telegram's getFile API to retrieve the audio, then downloads the .ogg voice file
- Sends the file to OpenAI Whisper for accurate multilingual transcription
- Passes the transcript to GPT-4o-mini which extracts 7 structured fields: title, description, priority, due date, project, tags, estimated time
- Creates a fully formatted Notion page with all metadata + the original transcript saved as a quote block
- Replies to the user in Telegram with a confirmation message and a clickable link to the new Notion task
- Cost: ~$0.002 per voice note

## How It Works

Node-by-node flow:

- **Telegram Trigger** — Listens for incoming messages from the bot
- **IF** — Checks if the message contains a voice attachment
- **Set** — Config block: Telegram token, OpenAI key, Notion token, database ID
- **HTTP Request (Telegram getFile)** — Retrieves file path from Telegram servers
- **HTTP Request (Telegram download)** — Downloads the raw .ogg audio file (binary)
- **HTTP Request (OpenAI Whisper)** — Transcribes voice to text via multipart form upload
- **Code** — Validates and extracts transcript, passes full config context
- **HTTP Request (OpenAI GPT-4o-mini)** — Extracts 7 task fields as structured JSON
- **Code** — Safely parses AI JSON with full fallback on parse error
- **HTTP Request (Notion API)** — Creates formatted Notion page with properties + content blocks
- **Telegram** — Sends success reply with task summary + Notion link
- **Telegram** — Sends "please send a voice message" reply for non-voice inputs
