## The Problem

1. 70% of online shopping carts are abandoned — $18 billion/year in lost Shopify revenue
2. Generic "You forgot something!" emails get \<2% recovery — customers ignore them instantly
3. Platforms like Klaviyo/Omnisend cost $100–$500/month for flows that still aren't personalized
4. Stores discount every cart equally — either killing margin on small orders or missing high-value conversions
5. No visibility into recovery attempts — no logging, no team alerts, no optimization possible

## The Solution

A Shopify webhook → n8n workflow that:
- Triggers on Shopify `abandoned_checkouts/create` webhook
- Waits 1 hour, then checks if the customer already completed the order (prevents spam)
- Uses GPT-4o-mini to generate a unique, warm recovery email based on customer name + exact cart items
- Automatically includes 10% discount code COMEBACK10 **only for carts $100+** — protecting margin on smaller orders
- Sends email via Gmail, logs to Google Sheets (11 columns), and fires a Slack alert for high-value carts

## How It Works

Node-by-node flow:

- **Webhook** — Receives Shopify abandoned_checkouts webhook (POST)
- **Code** — Extracts + normalizes checkout data
- **Wait** — 1 hour delay before follow-up
- **HTTP Request** — Shopify Orders API check (did they complete order?)
- **IF** — Skip if order already placed
- **OpenAI (gpt-4o-mini)** — Generates personalized recovery email as JSON
- **Code** — Parses AI response with fallback template
- **IF** — Routes high-value ($100+) vs. standard cart
- **Gmail (×2)** — Sends with or without discount based on cart value
- **Google Sheets** — Logs 11 columns per recovery attempt
- **IF + Slack** — Slack alert only for high-value carts
- **No Op (×2)** — Silent passes for already-ordered / non-Slack paths
- **Respond to Webhook** — Instant 200 OK to Shopify
