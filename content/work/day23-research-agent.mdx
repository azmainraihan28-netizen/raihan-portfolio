## The Problem

1. Writers spend 3–5 hours per article manually searching, reading, and synthesizing sources — most of it is tedious Google tab-switching
2. AI tools like ChatGPT hallucinate facts and statistics when writing research-backed content without real sources fed to them
3. Research quality is inconsistent across a team — different researchers find different information, producing wildly different content quality
4. Multi-step content workflows require 5+ separate tools (Google, Perplexity, Docs, Grammarly, Notion) — losing hours to context switching per piece
5. Content agencies and freelancers can't scale production without hiring more researchers at $20–$50/hour

## The Solution

A webhook-triggered n8n research agent that:
- Accepts any topic via POST request
- Uses GPT-4o-mini to generate a 5-angle research plan with targeted search queries for each angle
- Runs SplitInBatches loop — executes 5 real Google searches via SerpAPI and extracts top 3 results + People Also Ask per angle (15 real sources total)
- Aggregates all research into a structured context and passes it to GPT-4o-mini for a full, fact-grounded draft (~800–1,000 words)
- Creates a new Google Doc with the complete draft + source list automatically
- Logs 13 metadata fields to Google Sheets, building a permanent research history database
- Optional Slack notification with summary + clickable Doc link

## How It Works

Node-by-node flow:

- **Webhook** — POST /research-agent
- **Set** — Normalizes 9 input fields (topic, output_type, word_count, email, slack_enabled, sheet_id, serpapi_key, openai_key, timestamp)
- **IF** — Validates topic not empty → 400 error on missing
- **HTTP Request (OpenAI)** — GPT-4o-mini: generates 5-angle research outline as JSON
- **Code** — Parses outline, returns 5 separate items for loop (with full context passthrough)
- **SplitInBatches** — Loops through angles one at a time (batch size: 1)
- **HTTP Request (SerpAPI)** — Live Google search per angle (5 calls total)
- **Code** — Extracts top 3 organic results + PAA questions per search
- **Aggregate** — Collects all 5 angle results into single item after loop completes
- **Code** — Builds structured 1,500-char research context string from all angles
- **HTTP Request (OpenAI)** — GPT-4o-mini: writes full research-backed draft as JSON
- **Code** — Parses draft, handles fallback gracefully
- **HTTP Request (Google Docs)** — Creates new document with AI-generated title
- **HTTP Request (Google Docs batchUpdate)** — Inserts full draft + source attribution
- **Google Sheets** — Logs 13-column research job record
- **IF** — Routes to Slack or silent skip
- **Slack** — Rich block message with summary, key points, direct Doc link
- **No Op** — Silent pass when Slack disabled
- **Respond to Webhook (×2)** — Returns full JSON result (200) or validation error (400)
