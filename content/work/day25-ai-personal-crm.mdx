## The Problem

1. Professionals miss follow-ups on 40–60% of important contacts — deals and relationships die silently in overflowing inboxes
2. Manual CRM data entry consumes 5–6 hours/week — most salespeople fill it incorrectly, inconsistently, or skip it entirely
3. No memory between meetings — you arrive to calls asking "remind me, what did we last discuss?" — destroying trust
4. Enterprise CRM tools (Salesforce, HubSpot, Pipedrive) cost $50–$1,200/month and still require manual input to function
5. Email, LinkedIn DMs, and meetings live in silos — no single relationship intelligence view of any contact's full history

## The Solution

A Gmail-triggered n8n workflow that:
- Polls the Gmail Sent folder every 15 minutes and processes each outgoing email
- Filters out system emails, newsletters, and no-reply addresses automatically
- Sends email metadata + body excerpt to GPT-4o-mini for 15-field CRM intelligence extraction
- Extracts: contact name, company, relationship type, relationship strength, interaction type, summary, sentiment, follow-up required, next action, follow-up days, urgency, deal stage, estimated deal value, key topics, language
- Logs a 21-column record to a Google Sheets personal CRM database — building a permanent relationship history
- Fires a rich Slack block alert only for high-urgency follow-ups that need same-day action
- Cost: ~$0.001 per email analyzed (GPT-4o-mini)

## How It Works

Node-by-node flow:

- **Gmail Trigger** — Polls Sent folder every 15 minutes (label filter: SENT)
- **Set** — Normalizes 11 input fields with safe defaults (to_address, to_name, subject, body_snippet, sent_date, message_id, thread_id, openai_key, sheet_id, slack_enabled, slack_channel)
- **IF** — Filters system/no-reply emails via regex → silent skip for non-human addresses
- **HTTP Request (OpenAI)** — GPT-4o-mini with JSON mode: 15-field CRM intelligence report
- **Code** — Safely parses AI JSON, merges with email context, builds full 21-field CRM record
- **Google Sheets** — Appends 21-column CRM row (Timestamp → Language, all intelligence fields)
- **IF** — Routes high-urgency + follow-up-required + Slack-enabled interactions
- **Slack** — Rich block alert: contact card, deal stage, value, summary, next action, follow-up timeline
- **No Op (×2)** — Silent passes for low-priority emails and system-email skips

## Credentials & Setup

- Gmail OAuth2
- OpenAI API Key (passed inline in Set node — replace `YOUR_OPENAI_API_KEY`)
- Google Sheets OAuth2 (set Sheet ID in Set node — replace `YOUR_GOOGLE_SHEET_ID`)
- Slack OAuth2 (optional — for high-urgency follow-up alerts)
