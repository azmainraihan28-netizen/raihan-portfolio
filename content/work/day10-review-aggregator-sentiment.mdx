## The Problem

1. Businesses miss negative reviews for 3–5 days on average — damaging brand trust before anyone responds
2. Manually monitoring Google + Trustpilot daily wastes 30–60 minutes of ops time per business
3. Reputation tools like Birdeye, Podium, and ReviewTrackers cost $200–$500/month
4. Generic alerts give only a star rating — no context, no theme, no action guidance
5. Multi-location and multi-client businesses have no unified cross-platform dashboard

## The Solution

A scheduled n8n workflow that:
- Pulls fresh reviews from Google Business Profile and Trustpilot every morning at 9 AM
- Normalizes both review sources into a single unified schema
- Runs each review through GPT-4o-mini for 7-field sentiment enrichment
- Appends a 14-column row to a Google Sheets reputation dashboard
- Fires an instant rich Slack alert for every negative review with urgency level and action guidance

## How It Works

Node-by-node flow:

- **Schedule Trigger** — runs daily at 9 AM
- **HTTP Request (×2)** — Google Business Profile API + Trustpilot API
- **Code (×2)** — normalizes both formats into a unified schema
- **Merge** — combines both streams into one
- **SplitInBatches** — processes one review at a time
- **OpenAI (gpt-4o-mini)** — sentiment, score, theme, emotion, urgency, summary
- **Code** — safely parses AI JSON with error fallback
- **Google Sheets** — 14-column dashboard row per review
- **IF** — routes negative reviews to Slack alert
- **Slack** — rich alert with urgency, theme, emotion, action status
- **NoOp** — silent pass for positive/neutral reviews
